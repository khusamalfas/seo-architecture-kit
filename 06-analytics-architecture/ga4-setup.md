# GA4 Setup (High-Level)

## Steps
1. Create GA4 property and data stream.
2. Implement base tag via GTM or gtag.
3. Track key events:
   - page_view
   - scroll / engagement
   - form_submit / lead
   - click_to_call
   - click_to_whatsapp / messenger
   - booking_start / booking_complete
4. Configure conversions for primary goals (leads, bookings, purchases).
5. Link to Google Ads if used.
