# Technical SEO Checklist (High-Level)

- [ ] Correct HTTP status codes
- [ ] Clean redirect rules (no chains/loops)
- [ ] Canonicals set correctly
- [ ] Sitemaps valid and submitted
- [ ] robots.txt valid
- [ ] No accidental noindex on key pages
- [ ] Correct hreflang (if used)
- [ ] Core Web Vitals within targets
