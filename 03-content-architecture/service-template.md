# Service Page Template

## Sections
1. H1 – clear service name + qualifier
2. Intro – what the service solves, who it is for
3. Benefits / outcomes
4. How it works (steps)
5. Indications / when to use
6. Contraindications / limitations (if relevant)
7. Pricing / tariffs
8. FAQ
9. Call-to-action

## SEO Elements
- Title: `{Service} – {Brand} / {City}`
- H1: main service name
- Meta description: intent + CTA
- Internal links: category, related services, blog support content
- JSON-LD: Service / MedicalService (for healthcare)
