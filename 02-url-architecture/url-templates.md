# URL Templates

## Core Entities

### Services
Pattern:
- `/services/{service-slug}/`

### Categories
Pattern:
- `/category/{category-slug}/`

### Blog
Pattern:
- `/blog/{topic-slug}/{article-slug}/`

### Local / GEO
Pattern:
- `/region/{city-slug}/{service-slug}/`

Each template maps to:
- Page type
- Content template
- Schema template
- Internal linking rules
